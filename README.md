# Crafty Controller voor UmbrelOS

Deze app is een community-port van [Crafty Controller](https://craftycontrol.com) voor UmbrelOS.

## ✨ Features
- Beheer meerdere Minecraft servers (Java & Bedrock)
- Moderne webinterface met live console
- Automatische backups en server scheduling
- Ondersteunt Forge, Fabric, Paper, Spigot en meer
